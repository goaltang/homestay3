# 后端架构总览

> [!info] 后端概述
> 基于 Spring Boot 3.0.2 + Java 17 的民宿管理系统后端服务

## 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| **Spring Boot** | 3.0.2 | 核心框架 |
| **Java** | 17 | 开发语言 |
| **Spring Data JPA** | - | ORM 框架 |
| **Spring Security** | - | 安全认证 |
| **MySQL** | - | 关系数据库 |
| **Redis** | - | 缓存/会话 |
| **Redisson** | 3.21.0 | 分布式锁 |
| **Flyway** | - | 数据库迁移 |
| **JWT** | 0.12.3 | Token 认证 |
| **Lombok** | 1.18.24 | 代码简化 |
| **MapStruct** | 1.5.5 | DTO 映射 |
| **ECharts** | - | 支付集成 |
| **WebSocket** | - | 实时通信 |

## 项目结构

```
homestay-backend/
├── src/main/java/com/homestay3/homestaybackend/
│   ├── config/           # 配置类
│   ├── controller/       # 控制器层
│   ├── service/          # 服务层
│   │   ├── impl/        # 服务实现
│   │   └── gateway/     # 支付网关
│   ├── repository/       # 数据访问层
│   ├── entity/           # 实体类
│   ├── dto/              # 数据传输对象
│   ├── mapper/           # 对象映射
│   ├── model/            # 模型枚举
│   │   └── enums/
│   ├── security/         # 安全认证
│   ├── exception/        # 异常处理
│   ├── util/             # 工具类
│   └── HomestayBackendApplication.java
├── src/main/resources/
│   ├── application.yml     # 配置文件
│   └── db/migration/       # Flyway 迁移脚本
└── uploads/                # 文件上传目录
```

## 分层架构

```
┌─────────────────────────────────────┐
│         Controller 层               │  ← HTTP 接口
├─────────────────────────────────────┤
│         Service 层                  │  ← 业务逻辑
├─────────────────────────────────────┤
│         Repository 层               │  ← 数据访问
├─────────────────────────────────────┤
│         Entity / Database           │  ← 数据持久化
└─────────────────────────────────────┘
```

## 核心模块

### 业务模块
- 房源管理 (Homestay)
- 订单管理 (Order)
- 用户管理 (User)
- 评价管理 (Review)
- 设施管理 (Amenity)
- 支付管理 (Payment)

### 系统模块
- 安全认证 (Security)
- 系统配置 (SystemConfig)
- 日志管理 (Log)
- 通知系统 (Notification)
- 审核系统 (Audit)
- 争议处理 (Dispute)

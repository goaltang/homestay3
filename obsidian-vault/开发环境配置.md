# 开发环境配置

> [!tip] 配置指南
> 快速搭建 Homestay 项目开发环境

## 环境要求

- **Node.js** >= 18.0.0
- **npm** >= 9.0.0 或 **pnpm** >= 8.0.0
- **Git** >= 2.30.0

## 初始化步骤

### 1. 安装依赖

```bash
# 管理后台
cd homestay-admin
npm install

# 后端服务
cd homestay-backend
npm install

# 前端应用
cd homestay-front
npm install
```

### 2. 启动开发服务器

```bash
cd homestay-admin
npm run dev
```

默认访问: http://localhost:5173

## 常用命令

| 命令 | 说明 |
|------|------|
| `npm run dev` | 启动开发服务器 |
| `npm run build` | 构建生产版本 |
| `npm run preview` | 预览生产构建 |

## IDE 推荐

- VS Code + Vue 官方插件
- Vue DevTools 浏览器扩展
- ESLint + Prettier

# 通知与仪表盘

> [!info] 模块概述
> 通知中心和管理仪表盘为管理员提供实时消息推送和数据概览服务。

## 路由配置

| 路由路径 | 组件 | 标题 |
|---------|------|------|
| `/dashboard` | `dashboard/index.vue` | 系统首页 |
| `/notifications` | `notifications/index.vue` | 通知中心 |

## 功能清单

### 1. 系统首页 (Dashboard)
- 关键指标卡片展示
  - 今日订单数
  - 今日收入
  - 待审核数
  - 新用户数
- 快捷操作入口
- 最近活动流
- 图表概览

### 2. 通知中心
- 系统通知列表
- 通知类型筛选
- 未读标记
- 通知详情查看
- 批量标记已读

## 通知类型

```
🔔 订单通知
  - 新订单提醒
  - 订单取消通知
  - 退款通知

👤 用户通知
  - 新用户注册
  - 认证申请
  - 用户举报

⚠️ 系统通知
  - 系统维护
  - 异常告警
  - 配置变更
```

## API 接口

相关 API 文件：
- `api/dashboard.ts` - 仪表盘数据
- `api/notification.ts` - 通知管理

# 统计分析模块

> [!info] 模块概述
> 统计分析模块提供系统运营数据的可视化展示，帮助管理员了解业务状况。

## 路由配置

| 路由路径 | 组件 | 标题 |
|---------|------|------|
| `/statistics` | `statistics/index.vue` | 统计分析 |

## 功能清单

### 1. 数据概览
- 总房源数
- 总订单数
- 总用户数
- 总收入金额

### 2. 订单统计
- 订单趋势图
- 订单状态分布
- 预订渠道分析
- 热门房源排名

### 3. 用户统计
- 用户增长趋势
- 用户活跃度
- 用户地域分布
- 新用户转化率

### 4. 收入统计
- 收入趋势图
- 房源收入排名
- 收入来源分析
- 退款率统计

## 图表类型

使用 **ECharts** 实现：
- 📊 柱状图 - 订单/收入对比
- 📈 折线图 - 趋势分析
- 🥧 饼图 - 分布占比
- 🗺️ 地图 - 地域分布

## API 接口

相关 API 文件：
- `api/statistics.ts` - 统计数据接口
- `api/dashboard.ts` - 仪表盘数据

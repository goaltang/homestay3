# 用户管理模块

> [!info] 模块概述
> 用户管理模块负责系统用户的管理，包括用户信息查看、身份认证审核、用户状态管理等功能。

## 路由配置

| 路由路径 | 组件 | 标题 |
|---------|------|------|
| `/users` | `user/list.vue` | 用户列表 |
| `/verifications` | `user/verification.vue` | 身份认证审核 |

## 功能清单

### 1. 用户列表
- 查看所有注册用户
- 用户搜索（姓名、手机号、邮箱）
- 用户状态管理（正常、禁用）
- 用户详情查看

### 2. 用户详情
- 基本信息（姓名、头像、联系方式）
- 注册时间
- 订单历史
- 评价记录

### 3. 身份认证审核
- 查看认证申请列表
- 审核用户上传的证件
- 审核通过/拒绝
- 审核备注

## API 接口

相关 API 文件：
- `api/user.ts` - 用户管理接口
- `api/auth.ts` - 认证相关接口

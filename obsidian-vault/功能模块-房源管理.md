# 房源管理模块

> [!info] 模块概述
> 房源管理是民宿系统的核心模块，负责民宿房源的增删改查、分组管理、图片管理等功能。

## 路由配置

| 路由路径 | 组件 | 标题 |
|---------|------|------|
| `/homestays` | `homestay/list.vue` | 房源列表 |
| `/homestays/add` | `homestay/edit.vue` | 新增房源 |
| `/homestays/edit/:id` | `homestay/edit.vue` | 编辑房源 |
| `/groups` | `homestay/GroupManage.vue` | 房源分组管理 |

## 功能清单

### 1. 房源列表
- 展示所有民宿房源
- 支持搜索和筛选
- 房源状态管理（上架/下架）
- 快捷操作（编辑、删除、查看详情）

### 2. 新增/编辑房源
- 基本信息填写（名称、描述、价格、地址）
- 图片上传和管理
- 设施选择
- 房型设置
- 预订规则配置

### 3. 房源分组管理
- 创建和管理房源分组
- 批量归类房源
- 分组统计

## API 接口

相关 API 文件：
- `api/homestay.ts` - 房源基础操作
- `api/homestayImage.ts` - 图片管理
- `api/group.ts` - 分组管理
- `api/amenities.ts` - 设施关联

# 系统管理模块

> [!info] 模块概述
> 系统管理模块负责系统级配置和运维管理，包括系统配置、日志管理、公告管理、争议处理等功能。

## 路由配置

| 路由路径 | 组件 | 标题 |
|---------|------|------|
| `/system/config` | `system/Config.vue` | 系统配置 |
| `/system/logs` | `system/OperationLog.vue` | 操作日志 |
| `/system/login-logs` | `system/LoginLog.vue` | 登录日志 |
| `/system/announcements` | `system/Announcement.vue` | 公告管理 |
| `/system/disputes` | `system/DisputeManage.vue` | 争议管理 |

## 功能清单

### 1. 系统配置
- 基础参数设置
- 支付配置
- 短信/邮件配置
- 上传限制

### 2. 操作日志
- 管理员操作记录
- 操作类型筛选
- 时间范围查询
- 操作详情查看

### 3. 登录日志
- 登录记录查询
- 登录 IP 追踪
- 异常登录检测
- 登录设备信息

### 4. 公告管理
- 系统公告发布
- 公告编辑/删除
- 公告状态管理
- 推送范围设置

### 5. 争议管理
- 用户争议处理
- 订单争议
- 退款争议
- 争议仲裁

## API 接口

相关 API 文件：
- `api/systemConfig.ts` - 系统配置接口
- `api/loginLog.ts` - 登录日志接口
- `api/announcement.ts` - 公告管理接口
- `api/dispute.ts` - 争议处理接口
- `api/notification.ts` - 通知管理接口

## 数据结构

```typescript
// 系统配置
interface SystemConfig {
  id: number
  key: string
  value: string
  description: string
}

// 操作日志
interface OperationLog {
  id: number
  userId: number
  action: string
  target: string
  ip: string
  createdAt: string
}

// 公告
interface Announcement {
  id: number
  title: string
  content: string
  status: 'draft' | 'published' | 'archived'
  publishAt: string
}

// 争议
interface Dispute {
  id: number
  orderId: number
  complainantId: number
  respondentId: number
  type: string
  status: 'pending' | 'processing' | 'resolved' | 'closed'
  description: string
}
```

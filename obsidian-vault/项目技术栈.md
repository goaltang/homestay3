# 项目技术栈

> [!summary] 技术栈概览
> Homestay 项目采用现代化的前端技术栈

## 前端技术 (homestay-admin)

### 核心框架
- **Vue 3** - Composition API + `<script setup>`
- **TypeScript** - 静态类型检查
- **Vite** - 快速构建工具

### 状态与路由
- **Pinia** - 状态管理
- **Vue Router 4** - 路由管理

### 自动化插件
- **unplugin-auto-import** - API 自动导入
- **unplugin-vue-components** - 组件自动注册

## 技术优势

| 特性 | 优势 |
|------|------|
| TypeScript | 类型安全，减少运行时错误 |
| Composition API | 更好的逻辑复用 |
| Vite | 极速开发体验 |
| 自动导入 | 减少样板代码 |

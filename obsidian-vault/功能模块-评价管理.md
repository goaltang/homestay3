# 评价管理模块

> [!info] 模块概述
> 评价管理模块负责管理用户对民宿的评价内容，包括评价审核、回复、违规处理等功能。

## 路由配置

| 路由路径 | 组件 | 标题 |
|---------|------|------|
| `/reviews` | `review/list.vue` | 评价列表 |

## 功能清单

### 1. 评价列表
- 查看所有用户评价
- 按房源筛选评价
- 按评分筛选
- 按时间排序

### 2. 评价详情
- 评价内容
- 评分（1-5星）
- 评价图片
- 订单关联
- 回复内容

### 3. 评价管理
- 评价审核
- 官方回复
- 违规标记
- 评价隐藏

## API 接口

相关 API 文件：
- `api/review.ts` - 评价管理接口
- `api/violation.ts` - 违规处理接口

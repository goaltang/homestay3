# 后端安全认证

> [!info] 安全模块
> 基于 Spring Security + JWT 的认证授权系统

## 技术栈

| 组件 | 版本 | 用途 |
|------|------|------|
| **Spring Security** | - | 安全框架 |
| **JJWT** | 0.12.3 | JWT Token 生成与验证 |
| **Redis** | - | Token 存储/会话管理 |

## 认证流程

```
用户登录 → 提交用户名密码
    ↓
AuthenticationController 验证
    ↓
SecurityConfig 配置过滤
    ↓
JwtTokenProvider 生成 Token
    ↓
返回 JWT Token 给客户端
```

## 核心类

### SecurityConfig

```java
@Configuration
@EnableWebSecurity
public class SecurityConfig {
    // 配置安全过滤链
    // 设置认证端点
    // 配置权限规则
}
```

### JwtAuthenticationFilter

```java
// JWT 认证过滤器
// 拦截请求
// 解析 Token
// 设置认证上下文
```

### JwtTokenProvider

```java
// 生成 Token
String generateToken(Authentication authentication)

// 验证 Token
boolean validateToken(String token)

// 获取用户信息
String getUsernameFromToken(String token)
```

### CustomUserDetails

```java
// 用户详情封装
// 实现 UserDetails 接口
// 包含用户信息和权限
```

## 权限控制

```java
// 基于角色的访问控制
ROLE_USER    // 普通用户
ROLE_HOST    // 房东
ROLE_ADMIN   // 管理员
```

## Token 结构

```
Header:
  - alg: HS256
  - typ: JWT

Payload:
  - sub: 用户名
  - id: 用户ID
  - role: 用户角色
  - iat: 签发时间
  - exp: 过期时间
```

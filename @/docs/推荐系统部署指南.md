# 🚀 推荐系统部署指南

## 📋 部署前准备

### 环境要求

- **Java**: JDK 17+
- **Node.js**: 16+
- **Redis**: 6.0+
- **MySQL**: 8.0+
- **Maven**: 3.8+

### 必要服务检查

```bash
# 检查Redis服务

# 应返回 PONG

# 检查MySQL服务
mysql -u root -p -e "SELECT VERSION();"

# 检查Java版本
java -version
```

## 🔧 后端部署

### 1. 环境配置

#### 1.1 Redis 配置

```properties
# application.properties 或 application.yml
spring.data.redis.host=localhost
spring.data.redis.port=6379
spring.data.redis.database=0
spring.data.redis.password=
spring.data.redis.timeout=5000
spring.data.redis.lettuce.pool.max-active=8
spring.data.redis.lettuce.pool.max-idle=8
spring.data.redis.lettuce.pool.min-idle=0
spring.data.redis.lettuce.pool.max-wait=-1
```

#### 1.2 缓存配置

```properties
spring.cache.type=redis
spring.cache.redis.time-to-live=3600000
spring.cache.redis.cache-null-values=false
```

### 2. 构建部署

#### 2.1 Maven 构建

```bash
cd homestay-backend
mvn clean package -DskipTests
```

#### 2.2 启动应用

```bash
java -jar target/homestay-backend-0.0.1-SNAPSHOT.jar
```

#### 2.3 验证部署

```bash
# 检查应用启动状态
curl http://localhost:8080/actuator/health

# 检查推荐接口
curl http://localhost:8080/api/recommendations/popular?limit=3
```

### 3. 数据库初始化

#### 3.1 添加推荐相关索引

```sql
-- 添加复合索引提升查询性能
CREATE INDEX idx_homestay_status_created ON homestay(status, created_at);
CREATE INDEX idx_homestay_location_status ON homestay(province_code, city_code, status);
CREATE INDEX idx_order_homestay_created ON orders(homestay_id, created_at);
CREATE INDEX idx_review_homestay_created ON review(homestay_id, created_at);

-- 添加评分相关索引
CREATE INDEX idx_homestay_rating ON homestay(average_rating);
CREATE INDEX idx_homestay_price ON homestay(price_per_night);
```

#### 3.2 缓存预热脚本

```sql
-- 可选：创建预热存储过程
DELIMITER //
CREATE PROCEDURE WarmupRecommendationCache()
BEGIN
    -- 预计算热门民宿数据
    SELECT h.* FROM homestay h
    WHERE h.status = 'ACTIVE'
    ORDER BY h.average_rating DESC, h.created_at DESC
    LIMIT 20;

    -- 预计算推荐民宿数据
    SELECT h.* FROM homestay h
    WHERE h.status = 'ACTIVE' AND h.featured = true
    ORDER BY h.average_rating DESC
    LIMIT 20;
END //
DELIMITER ;
```

## 🔄 备份和恢复

### 1. Redis 备份

```bash
# 手动备份
redis-cli bgsave
cp /var/lib/redis/dump.rdb /backup/redis_backup_$(date +%Y%m%d_%H%M%S).rdb

# 定时备份脚本
#!/bin/bash
BACKUP_DIR="/backup/redis"
DATE=$(date +%Y%m%d_%H%M%S)
redis-cli bgsave
sleep 10
cp /var/lib/redis/dump.rdb "$BACKUP_DIR/dump_$DATE.rdb"
find $BACKUP_DIR -name "dump_*.rdb" -mtime +7 -delete
```

### 2. 配置备份

```bash
# 备份应用配置
cp application.properties /backup/config/application.properties.$(date +%Y%m%d)

# 备份nginx配置
cp /etc/nginx/sites-available/homestay /backup/nginx/homestay.$(date +%Y%m%d)
```

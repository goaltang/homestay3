## 📊 数据流程优化

### 页面加载时

1. **初始化** → 检查登录状态
2. **数据加载** → 并行加载房源列表和统计数据
3. **启动定时器** → 开始自动刷新统计数据

### 用户操作时

1. **手动刷新** → 全量数据刷新 + 加载动画
2. **审核操作** → 操作完成后自动刷新数据
3. **自动刷新** → 后台静默更新统计数据

### 页面卸载时

- **清理定时器** → 防止内存泄漏

## 🔄 生命周期管理

```javascript
onMounted(() => {
  refreshData(); // 初始化数据
  startAutoRefresh(); // 启动自动刷新
});

onUnmounted(() => {
  stopAutoRefresh(); // 清理定时器
});
```

## 📝 使用说明

### 统计面板

- **自动更新**：每分钟自动刷新一次
- **手动刷新**：点击"手动刷新"按钮立即更新
- **控制开关**：可关闭自动刷新功能

### 快速操作

- **审核统计报表**：查看详细数据分析（跳转专门页面）
- **审核历史记录**：查看所有审核记录（跳转专门页面）
- **通知中心**：查看系统通知

### 审核流程

- **单个审核**：点击房源卡片进入详细审核
- **批量审核**：选择多个房源后批量操作
- **快速操作**：快速通过/拒绝功能

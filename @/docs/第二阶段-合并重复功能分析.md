# 第二阶段：合并重复功能分析

## 现状分析

### 发现的重复功能

#### 1. 价格范围设置重复

**搜索栏高级搜索中的价格设置：**

```vue
<div class="advanced-item">
  <div class="label">价格范围</div>
  <div class="price-range">
    <el-input-number v-model="searchParams.minPrice" :min="0" :step="50" placeholder="最低价"
      controls-position="right" style="width: 110px;" />
    <span class="separator">-</span>
    <el-input-number v-model="searchParams.maxPrice" :min="0" :step="50" placeholder="最高价"
      controls-position="right" style="width: 110px;" />
  </div>
</div>
```

**筛选栏中的价格设置：**

```vue
<div class="filter-section">
  <h4>价格范围</h4>
  <div class="price-range">
    <el-input-number v-model="searchParams.minPrice" :min="0" :step="100" placeholder="最低价"
      controls-position="right" style="width: 110px;" />
    <span class="separator">-</span>
    <el-input-number v-model="searchParams.maxPrice" :min="0" :step="100" placeholder="最高价"
      controls-position="right" style="width: 110px;" />
  </div>
</div>
```

**问题：**

- 两处都有相同的价格范围设置功能
- 步长不同（50 vs 100），用户体验不一致
- 占用了重复的屏幕空间

#### 2. 重置功能重复

**搜索栏的重置：**

```typescript
const resetSearchParams = () => {
  // 重置搜索条件
  searchParams.selectedRegion = [];
  searchParams.keyword = "";
  searchParams.checkIn = null;
  searchParams.checkOut = null;
  searchParams.minPrice = null;
  searchParams.maxPrice = null;

  // 重置客人数量
  guestCounts.adults = 1;
  guestCounts.children = 0;
  guestCounts.infants = 0;
  searchParams.guestCount = 1;

  // 关闭相关弹窗
  showGuestSelector.value = false;
  showSuggestions.value = false;
  showAdvancedSearch.value = false;

  ElMessage.success("搜索条件已重置");
};
```

**筛选栏的重置：**

```typescript
const resetFilters = () => {
  searchParams.minPrice = null;
  searchParams.maxPrice = null;
  searchParams.amenities = [];
};
```

**问题：**

- 两个重置功能职责重叠
- `resetFilters` 只重置部分条件，功能不完整
- 用户不明确应该点击哪个重置按钮

#### 3. 功能分散问题

**当前状态：**

- 搜索栏：地区、日期、人数、关键词、价格
- 筛选栏：房源类型、价格、设施

**问题：**

- 价格范围在两处都可以设置，容易混淆
- 用户需要在多个地方查找功能
- 界面显得重复和冗余

## 解决方案设计

### 策略 1：功能职责明确划分

#### 搜索栏专注于"搜索"

- **地区选择**：目的地搜索
- **日期选择**：入住/退房日期
- **人数选择**：房客数量
- **关键词搜索**：名称、特色等文本搜索
- **移除**：价格范围设置

#### 筛选栏专注于"筛选"

- **房源类型筛选**：公寓、别墅、民宿等
- **价格范围筛选**：统一的价格设置
- **设施筛选**：WiFi、空调、厨房等
- **其他筛选条件**：评分、距离等

### 策略 2：重置功能统一

#### 合并重置逻辑

```typescript
const resetAllConditions = () => {
  // 重置所有搜索和筛选条件
  searchParams.selectedRegion = [];
  searchParams.keyword = "";
  searchParams.checkIn = null;
  searchParams.checkOut = null;
  searchParams.guestCount = 1;
  searchParams.minPrice = null;
  searchParams.maxPrice = null;
  searchParams.propertyType = null;
  searchParams.amenities = [];

  // 重置客人数量
  guestCounts.adults = 1;
  guestCounts.children = 0;
  guestCounts.infants = 0;

  // 关闭相关弹窗
  showGuestSelector.value = false;
  showSuggestions.value = false;
  showAdvancedSearch.value = false;

  ElMessage.success("所有条件已重置");
};
```

#### 分级重置功能

- **搜索栏重置**：只重置搜索相关条件（地区、日期、人数、关键词）
- **筛选栏重置**：只重置筛选相关条件（类型、价格、设施）
- **全部重置**：重置所有条件

### 策略 3：界面优化

#### 移除搜索栏中的价格设置

- 从高级搜索选项中移除价格范围
- 统一在筛选栏中设置价格
- 减少功能重复

#### 优化重置按钮位置

- 搜索栏：保留重置按钮，但只重置搜索条件
- 筛选栏：保留重置按钮，重置筛选条件
- 增加全局"清除所有"按钮

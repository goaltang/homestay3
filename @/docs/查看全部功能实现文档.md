# 查看全部功能实现文档

## 实现概述

查看全部功能是第一阶段优化的重要组成部分，为用户提供了从首页跳转到完整列表页面的能力。

## 功能分类

### 1. 浏览模式下的查看全部

#### 1.1 查看全部推荐民宿

- **触发位置**: 推荐民宿区域的"查看全部"按钮
- **路由跳转**: `/homestays?featured=true`
- **实现方法**: `viewAllFeatured()`

#### 1.2 查看全部最新上架

- **触发位置**: 最新上架区域的"查看全部"按钮
- **路由跳转**: `/homestays?status=ACTIVE`
- **实现方法**: `viewAllActive()`

### 2. 搜索模式下的查看全部

#### 2.1 查看全部搜索结果

- **触发位置**: 搜索结果区域的"查看全部搜索结果"按钮
- **路由跳转**: `/homestays` + 完整搜索条件参数
- **实现方法**: `viewAllSearchResults()`

#### 2.1 基础查看全部方法

```typescript
const viewAllFeatured = () => {
  router.push("/homestays?featured=true");
};

const viewAllActive = () => {

```

#### 2.2 搜索结果查看全部方法

```typescript
const viewAllSearchResults = () => {
  const query: any = {};

  // 构建包含当前所有搜索条件的查询参数
  if (searchParams.selectedRegion.length > 0) {
    query.region = searchParams.selectedRegion.join(",");
  }

  if (searchParams.keyword.trim()) {
    query.keyword = searchParams.keyword.trim();
  }

  if (searchParams.propertyType) {
    query.type = searchParams.propertyType;
  }

  if (searchParams.checkIn) {
    query.checkIn = searchParams.checkIn;
  }

  if (searchParams.checkOut) {
    query.checkOut = searchParams.checkOut;
  }

  if (searchParams.guestCount > 1) {
    query.guests = searchParams.guestCount;
  }

  if (searchParams.minPrice !== null) {
    query.minPrice = searchParams.minPrice.toString();
  }

  if (searchParams.maxPrice !== null) {
    query.maxPrice = searchParams.maxPrice.toString();
  }

  if (searchParams.amenities.length > 0) {
    query.amenities = searchParams.amenities.join(",");
  }

  router.push({ path: "/homestays", query });
};
```

## 目标页面优化

### 1. HomestayListView.vue 增强

#### 1.1 动态页面标题

```typescript
const pageTitle = computed(() => {
  if (route.query.featured) return "精选民宿";
  if (route.query.status === "ACTIVE") return "最新上架";
  if (route.query.keyword) return `"${route.query.keyword}" 的搜索结果`;
  if (route.query.region) return `${route.query.region} 的民宿`;
  return "全部民宿";
});
```

## 用户体验流程

### 1. 浏览模式流程

1. 用户在首页浏览推荐民宿
2. 点击"查看全部"按钮
3. 跳转到民宿列表页，显示"精选民宿"
4. 可以点击"返回首页"回到首页

### 2. 搜索模式流程

1. 用户在首页设置搜索条件
2. 查看搜索结果（前 3-6 个）
3. 点击"查看全部搜索结果"按钮
4. 跳转到民宿列表页，保持所有搜索条件
5. 显示动态标题如"杭州 的搜索结果"
6. 可以在列表页进一步筛选

### 3. 最新上架流程

1. 用户在首页浏览最新上架区域
2. 点击"查看全部"按钮
3. 跳转到民宿列表页，显示"最新上架"
4. 展示所有最新上架的民宿

# 支付系统改造详细步骤指南

## 🎯 总体流程概览

```
环境准备 → 基础设施 → 支付宝集成 → 微信集成 → 系统集成 → 测试验证
   您      我         我          我         我         双方
 0.5天    1天       1.5天      1.5天      1天        2天
```

## 📋 第一阶段：环境准备 (您来完成 - 0.5 天)

### 1.1 支付宝沙箱环境申请

#### 步骤 1：注册蚂蚁金服开放平台账号

1. 访问：https://openhome.alipay.com/
2. 点击"立即入驻" → 选择"自研开发者"
3. 使用手机号注册并完成实名认证
4. 登录后进入开发者中心

#### 步骤 2：创建应用

1. 进入"开发者中心" → "我的应用"
2. 点击"创建应用" → 选择"网页&移动应用"
3. 填写应用信息：
   - 应用名称：`民宿系统测试应用`
   - 应用简介：`民宿预订系统支付功能测试`
   - 应用类型：选择"自用型应用"

#### 步骤 3：配置应用信息

1. 应用创建成功后，点击"查看"进入应用详情
2. 记录下 **APPID**（重要）
3. 配置应用网关：暂时可以填写 `http://localhost:8080`

#### 步骤 4：生成 RSA 密钥

1. 下载支付宝提供的密钥生成工具：

   - Windows：https://ideservice.alipay.com/ide/getPluginUrl.htm?clientType=assistant&platform=win&channelType=WEB
   - Mac：https://ideservice.alipay.com/ide/getPluginUrl.htm?clientType=assistant&platform=mac&channelType=WEB

2. 运行密钥生成工具：

   - 选择密钥长度：`2048`
   - 选择密钥格式：`PKCS1`
   - 点击"生成密钥"

3. 保存生成的密钥：
   - **应用私钥**（重要）：复制并保存到安全位置
   - **应用公钥**：需要上传到支付宝

#### 步骤 5：上传公钥并获取支付宝公钥

1. 在应用详情页面，找到"开发设置"
2. 点击"接口加签方式" → "公钥"
3. 将生成的**应用公钥**粘贴并保存
4. 保存后，复制**支付宝公钥**（重要）

#### 步骤 6：开通沙箱环境

1. 在应用详情页面，点击"沙箱环境"
2. 获取沙箱信息：
   - **沙箱 APPID**
   - **沙箱网关地址**：`https://openapi.alipaydev.com/gateway.do`

#### 步骤 7：下载沙箱版支付宝 APP

1. 扫描沙箱页面的二维码下载"沙箱版支付宝"
2. 使用沙箱账号登录（页面会提供测试账号）

### 1.2 微信支付沙箱环境申请

#### 步骤 1：注册微信商户平台

1. 访问：https://pay.weixin.qq.com/
2. 点击"接入微信支付"
3. 选择主体类型（个人选择"小微商户"）
4. 填写商户信息并提交审核

#### 步骤 2：获取开发配置

审核通过后：

1. 登录微信商户平台
2. 进入"开发配置" → "开发配置"
3. 记录以下信息：
   - **商户号(mch_id)**
   - **APPID** (需要关联小程序或公众号)

#### 步骤 3：设置 API 密钥

1. 在"开发配置"中点击"API 安全"
2. 设置**API 密钥**(32 位)：建议使用随机字符串
3. 记录下设置的 API 密钥（重要）

#### 步骤 4：下载 API 证书

1. 在"API 安全"页面，下载"API 证书"
2. 解压后得到三个文件：
   - `apiclient_cert.p12`（重要）
   - `apiclient_cert.pem`
   - `apiclient_key.pem`

#### 步骤 5：申请沙箱环境

1. 访问微信支付沙箱环境：https://pay.weixin.qq.com/wiki/doc/api/jsapi.php?chapter=23_1
2. 使用商户平台账号登录
3. 获取沙箱配置信息

### 3.2 支付宝网关实现

#### AlipayGateway.java

```java
@Component
@Slf4j
public class AlipayGateway implements PaymentGateway {

    private final PaymentConfig paymentConfig;
    private final AlipayClient alipayClient;

    public AlipayGateway(PaymentConfig paymentConfig) {
        this.paymentConfig = paymentConfig;
        this.alipayClient = new DefaultAlipayClient(
            paymentConfig.getAlipay().getGatewayUrl(),
            paymentConfig.getAlipay().getAppId(),
            paymentConfig.getAlipay().getPrivateKey(),
            "json",
            "UTF-8",
            paymentConfig.getAlipay().getPublicKey(),
            "RSA2"
        );
    }

    @Override
    public PaymentResponse createPayment(PaymentRequest request) {
        try {
            // 创建支付宝当面付请求
            AlipayTradePrecreateRequest alipayRequest = new AlipayTradePrecreateRequest();
            alipayRequest.setNotifyUrl(paymentConfig.getAlipay().getNotifyUrl());

            // 构建请求参数
            Map<String, Object> bizContent = new HashMap<>();
            bizContent.put("out_trade_no", request.getOutTradeNo());
            bizContent.put("total_amount", request.getAmount().toString());
            bizContent.put("subject", request.getSubject());
            bizContent.put("store_id", "test_store");
            bizContent.put("timeout_express", "10m");

            alipayRequest.setBizContent(JSON.toJSONString(bizContent));

            // 发起请求
            AlipayTradePrecreateResponse response = alipayClient.execute(alipayRequest);

            if (response.isSuccess()) {
                return PaymentResponse.builder()
                    .success(true)
                    .qrCode(response.getQrCode())
                    .outTradeNo(request.getOutTradeNo())
                    .message("二维码生成成功")
                    .build();
            } else {
                log.error("支付宝创建订单失败: {}", response.getSubMsg());
                return PaymentResponse.builder()
                    .success(false)
                    .message(response.getSubMsg())
                    .build();
            }
        } catch (Exception e) {
            log.error("支付宝创建支付订单异常", e);
            return PaymentResponse.builder()
                .success(false)
                .message("创建支付订单失败: " + e.getMessage())
                .build();
        }
    }

    @Override
    public PaymentStatusResponse queryPayment(String outTradeNo) {
        try {
            AlipayTradeQueryRequest request = new AlipayTradeQueryRequest();
            Map<String, Object> bizContent = new HashMap<>();
            bizContent.put("out_trade_no", outTradeNo);
            request.setBizContent(JSON.toJSONString(bizContent));

            AlipayTradeQueryResponse response = alipayClient.execute(request);

            if (response.isSuccess()) {
                String tradeStatus = response.getTradeStatus();
                PaymentStatus status = convertAlipayStatus(tradeStatus);

                return PaymentStatusResponse.builder()
                    .success(true)
                    .outTradeNo(outTradeNo)
                    .transactionId(response.getTradeNo())
                    .status(status)
                    .amount(new BigDecimal(response.getTotalAmount()))
                    .message("查询成功")
                    .build();
            } else {
                return PaymentStatusResponse.builder()
                    .success(false)
                    .message(response.getSubMsg())
                    .build();
            }
        } catch (Exception e) {
            log.error("查询支付宝订单状态异常", e);
            return PaymentStatusResponse.builder()
                .success(false)
                .message("查询失败: " + e.getMessage())
                .build();
        }
    }

    @Override
    public boolean verifyNotify(Map<String, String> params) {
        try {
            return AlipaySignature.rsaCheckV1(
                params,
                paymentConfig.getAlipay().getPublicKey(),
                "UTF-8",
                "RSA2"
            );
        } catch (Exception e) {
            log.error("验证支付宝回调签名失败", e);
            return false;
        }
    }

    // 其他方法实现...
}
```

### 3.3 支付宝回调处理

#### AlipayCallbackController.java

```java
@RestController
@RequestMapping("/api/payment/alipay")
@Slf4j
public class AlipayCallbackController {

    private final AlipayGateway alipayGateway;
    private final PaymentService paymentService;

    @PostMapping("/notify")
    public String notify(HttpServletRequest request) {
        try {
            // 获取回调参数
            Map<String, String> params = new HashMap<>();
            Map<String, String[]> requestParams = request.getParameterMap();
            for (String name : requestParams.keySet()) {
                String[] values = requestParams.get(name);
                params.put(name, values[0]);
            }

            log.info("收到支付宝回调: {}", params);

            // 验证签名
            if (!alipayGateway.verifyNotify(params)) {
                log.error("支付宝回调签名验证失败");
                return "fail";
            }

            // 处理回调
            PaymentNotifyResult result = alipayGateway.handleNotify(params);
            if (result.isSuccess()) {
                // 更新订单状态
                paymentService.handlePaymentNotify(result);
                return "success";
            } else {
                log.error("处理支付宝回调失败: {}", result.getMessage());
                return "fail";
            }

        } catch (Exception e) {
            log.error("处理支付宝回调异常", e);
            return "fail";
        }
    }
}
```

### 4.1 微信支付网关实现

#### WechatGateway.java

```java
@Component
@Slf4j
public class WechatGateway implements PaymentGateway {

    private final PaymentConfig paymentConfig;
    private final CloseableHttpClient httpClient;

    public WechatGateway(PaymentConfig paymentConfig) {
        this.paymentConfig = paymentConfig;
        this.httpClient = createHttpClient();
    }

    @Override
    public PaymentResponse createPayment(PaymentRequest request) {
        try {
            String url = "https://api.mch.weixin.qq.com/pay/unifiedorder";

            // 构建请求参数
            Map<String, String> params = new HashMap<>();
            params.put("appid", paymentConfig.getWechat().getAppId());
            params.put("mch_id", paymentConfig.getWechat().getMchId());
            params.put("nonce_str", generateNonceStr());
            params.put("body", request.getSubject());
            params.put("out_trade_no", request.getOutTradeNo());
            params.put("total_fee", request.getAmount().multiply(new BigDecimal("100")).intValue() + "");
            params.put("spbill_create_ip", "127.0.0.1");
            params.put("notify_url", paymentConfig.getWechat().getNotifyUrl());
            params.put("trade_type", "NATIVE");

            // 生成签名
            String sign = generateSign(params);
            params.put("sign", sign);

            // 构建XML请求
            String xmlRequest = mapToXml(params);
            log.info("微信支付请求: {}", xmlRequest);

            // 发送请求
            HttpPost httpPost = new HttpPost(url);
            httpPost.setEntity(new StringEntity(xmlRequest, "UTF-8"));
            httpPost.setHeader("Content-Type", "application/xml; charset=UTF-8");

            CloseableHttpResponse response = httpClient.execute(httpPost);
            String responseBody = EntityUtils.toString(response.getEntity(), "UTF-8");

            log.info("微信支付响应: {}", responseBody);

            // 解析响应
            Map<String, String> responseMap = xmlToMap(responseBody);

            if ("SUCCESS".equals(responseMap.get("return_code")) &&
                "SUCCESS".equals(responseMap.get("result_code"))) {

                return PaymentResponse.builder()
                    .success(true)
                    .qrCode(responseMap.get("code_url"))
                    .outTradeNo(request.getOutTradeNo())
                    .message("二维码生成成功")
                    .build();
            } else {
                String errorMsg = responseMap.get("return_msg") != null ?
                    responseMap.get("return_msg") : responseMap.get("err_code_des");

                return PaymentResponse.builder()
                    .success(false)
                    .message(errorMsg)
                    .build();
            }

        } catch (Exception e) {
            log.error("微信创建支付订单异常", e);
            return PaymentResponse.builder()
                .success(false)
                .message("创建支付订单失败: " + e.getMessage())
                .build();
        }
    }

    private CloseableHttpClient createHttpClient() {
        try {
            // 加载证书
            KeyStore keyStore = KeyStore.getInstance("PKCS12");
            InputStream certStream = this.getClass().getClassLoader()
                .getResourceAsStream("certs/apiclient_cert.p12");
            keyStore.load(certStream, paymentConfig.getWechat().getMchId().toCharArray());

            // 创建SSL上下文
            SSLContext sslContext = SSLContexts.custom()
                .loadKeyMaterial(keyStore, paymentConfig.getWechat().getMchId().toCharArray())
                .build();

            // 创建HTTP客户端
            return HttpClients.custom()
                .setSSLContext(sslContext)
                .build();

        } catch (Exception e) {
            log.error("创建微信支付HTTP客户端失败", e);
            throw new RuntimeException("微信支付初始化失败", e);
        }
    }

    // 其他辅助方法...
}
```

### 4.2 微信支付回调处理

#### WechatCallbackController.java

```java
@RestController
@RequestMapping("/api/payment/wechat")
@Slf4j
public class WechatCallbackController {

    private final WechatGateway wechatGateway;
    private final PaymentService paymentService;

    @PostMapping("/notify")
    public String notify(HttpServletRequest request) {
        try {
            // 读取回调数据
            BufferedReader reader = request.getReader();
            StringBuilder sb = new StringBuilder();
            String line;
            while ((line = reader.readLine()) != null) {
                sb.append(line);
            }
            String xmlData = sb.toString();

            log.info("收到微信支付回调: {}", xmlData);

            // 解析XML数据
            Map<String, String> params = xmlToMap(xmlData);

            // 验证签名
            if (!wechatGateway.verifyNotify(params)) {
                log.error("微信支付回调签名验证失败");
                return buildFailXml("签名验证失败");
            }

            // 处理回调
            PaymentNotifyResult result = wechatGateway.handleNotify(params);
            if (result.isSuccess()) {
                // 更新订单状态
                paymentService.handlePaymentNotify(result);
                return buildSuccessXml();
            } else {
                log.error("处理微信支付回调失败: {}", result.getMessage());
                return buildFailXml(result.getMessage());
            }

        } catch (Exception e) {
            log.error("处理微信支付回调异常", e);
            return buildFailXml("系统异常");
        }
    }

    private String buildSuccessXml() {
        return "<xml><return_code><![CDATA[SUCCESS]]></return_code>" +
               "<return_msg><![CDATA[OK]]></return_msg></xml>";
    }

    private String buildFailXml(String msg) {
        return "<xml><return_code><![CDATA[FAIL]]></return_code>" +
               "<return_msg><![CDATA[" + msg + "]]></return_msg></xml>";
    }
}
```

### 5.1 重构 PaymentServiceImpl

```java
@Service
@Transactional
@Slf4j
public class PaymentServiceImpl implements PaymentService {

    private final OrderRepository orderRepository;
    private final PaymentRecordRepository paymentRecordRepository;
    private final AlipayGateway alipayGateway;
    private final WechatGateway wechatGateway;

    @Override
    public String generatePaymentQRCode(Long orderId, String method) {
        try {
            // 查询订单
            Order order = orderRepository.findById(orderId)
                .orElseThrow(() -> new ResourceNotFoundException("订单不存在"));

            // 生成商户订单号
            String outTradeNo = generateOutTradeNo(orderId);

            // 创建支付请求
            PaymentRequest request = PaymentRequest.builder()
                .outTradeNo(outTradeNo)
                .amount(order.getTotalPrice())
                .subject("民宿预订-" + order.getHomestay().getTitle())
                .orderId(orderId)
                .build();

            // 选择支付网关
            PaymentGateway gateway = selectPaymentGateway(method);

            // 创建支付
            PaymentResponse response = gateway.createPayment(request);

            if (response.isSuccess()) {
                // 保存支付记录
                savePaymentRecord(orderId, method, outTradeNo, request, response);

                // 更新订单状态
                order.setStatus(OrderStatus.PAYMENT_PENDING.name());
                orderRepository.save(order);

                return response.getQrCode();
            } else {
                throw new RuntimeException("生成支付二维码失败: " + response.getMessage());
            }

        } catch (Exception e) {
            log.error("生成支付二维码异常", e);
            throw new RuntimeException("生成支付二维码失败", e);
        }
    }

    @Override
    public boolean checkPaymentStatus(Long orderId) {
        try {
            // 查询最新的支付记录
            PaymentRecord paymentRecord = paymentRecordRepository
                .findTopByOrderIdOrderByCreatedAtDesc(orderId)
                .orElse(null);

            if (paymentRecord == null) {
                return false;
            }

            // 选择支付网关查询状态
            PaymentGateway gateway = selectPaymentGateway(paymentRecord.getPaymentMethod());
            PaymentStatusResponse response = gateway.queryPayment(paymentRecord.getOutTradeNo());

            if (response.isSuccess() && response.getStatus() == PaymentStatus.PAID) {
                // 更新支付记录
                paymentRecord.setStatus("SUCCESS");
                paymentRecord.setTransactionId(response.getTransactionId());
                paymentRecordRepository.save(paymentRecord);

                // 更新订单状态
                updateOrderPaymentStatus(orderId, PaymentStatus.PAID);
                return true;
            }

            return false;

        } catch (Exception e) {
            log.error("检查支付状态异常", e);
            return false;
        }
    }

    public void handlePaymentNotify(PaymentNotifyResult result) {
        try {
            // 查找支付记录
            PaymentRecord paymentRecord = paymentRecordRepository
                .findByOutTradeNo(result.getOutTradeNo())
                .orElseThrow(() -> new RuntimeException("支付记录不存在"));

            // 防重复处理
            if ("SUCCESS".equals(paymentRecord.getStatus())) {
                log.info("支付记录已处理，跳过: {}", result.getOutTradeNo());
                return;
            }

            // 更新支付记录
            paymentRecord.setStatus("SUCCESS");
            paymentRecord.setTransactionId(result.getTransactionId());
            paymentRecord.setNotifyParams(JSON.toJSONString(result.getParams()));
            paymentRecordRepository.save(paymentRecord);

            // 更新订单状态
            updateOrderPaymentStatus(paymentRecord.getOrderId(), PaymentStatus.PAID);

            log.info("支付回调处理成功: orderId={}, outTradeNo={}",
                paymentRecord.getOrderId(), result.getOutTradeNo());

        } catch (Exception e) {
            log.error("处理支付回调异常", e);
            throw new RuntimeException("处理支付回调失败", e);
        }
    }

    private PaymentGateway selectPaymentGateway(String method) {
        switch (method.toLowerCase()) {
            case "alipay":
                return alipayGateway;
            case "wechat":
                return wechatGateway;
            default:
                throw new IllegalArgumentException("不支持的支付方式: " + method);
        }
    }

    // 其他辅助方法...
}
```

## API 测试结果

### 1. 地图搜索 API (`/api/homestays/map-search`)
```bash
curl -X POST http://localhost:8080/api/homestays/map-search \
  -H "Content-Type: application/json" \
  -d '{
    "northEastLat": 30.3,
    "northEastLng": 120.2,
    "southWestLat": 30.2,
    "southWestLng": 120.1
  }'
```
**结果**: ✅ 返回杭州西湖区域房源数据
- 返回房源数量: 10+ 条
- 包含字段: id, title, price, latitude, longitude 等

---

### 2. 聚合点 API (`/api/homestays/map-clusters`)
```bash
curl -X POST http://localhost:8080/api/homestays/map-clusters \
  -H "Content-Type: application/json" \
  -d '{
    "northEastLat": 30.3,
    "northEastLng": 120.2,
    "southWestLat": 30.2,
    "southWestLng": 120.1,
    "zoom": 12
  }'
```
**结果**: ✅ 返回4个聚合点
```json
[
  {"latitude": 30.261099, "longitude": 120.120322, "count": 5},
  {"latitude": 30.240241, "longitude": 120.171809, "count": 3},
  {"latitude": 30.230465, "longitude": 120.133063, "count": 2},
  {"latitude": 30.23436, "longitude": 120.167959, "count": 1}
]
```
**分析**: 
- 聚合算法正常工作
- 根据 zoom=12 动态计算网格大小
- 按 count 降序排列

---

### 3. 附近房源 API (`/api/homestays/nearby`)
```bash
curl -X POST http://localhost:8080/api/homestays/nearby \
  -H "Content-Type: application/json" \
  -d '{
    "latitude": 30.2485,
    "longitude": 120.1433,
    "radiusKm": 5,
    "sortBy": "DISTANCE",
    "limit": 3
  }'
```
**结果**: ✅ 返回3个房源，按距离排序
```json
[
  {"id": 107, "distanceKm": 0.44, ...},
  {"distanceKm": 1.11, ...},
  {"distanceKm": 2.14, ...}
]
```
**分析**:
- Haversine 距离计算正常
- 距离排序正确（升序）
- 距离精度保留2位小数

## 使用示例

### 在代码中调用
```typescript
const { searchNearby, searchByLandmark, loadClusters } = useMapSearch()

// 1. 搜索附近5km内的房源
await searchNearby({ radius: 5 })

// 2. 搜索地标周边（西湖3km内）
await searchByLandmark(30.2485, 120.1433, 3)

// 3. 加载聚合点
await loadClusters()
```

### 用户操作
1. 访问 `http://localhost:5173/map-search`
2. 点击"附近"按钮 → 搜索当前位置周边
3. 调整半径滑块 → 重新搜索
4. 点击"聚合"按钮 → 查看房源分布密度
5. 拖动地图 → 自动刷新房源

# 开发环境配置指南

> [!tip] 配置指南
> 本文档帮助你快速搭建 Homestay 项目的开发环境

## 环境要求

### 必需软件
- **Node.js** >= 18.0.0
- **npm** >= 9.0.0 或 **pnpm** >= 8.0.0
- **Git** >= 2.30.0

### 推荐工具
- **VS Code** - 推荐的代码编辑器
- **Vue DevTools** - 浏览器扩展
- **Git** - 版本控制

## 项目初始化

### 1. 克隆项目
```bash
git clone <repository-url>
cd homestay3
```

### 2. 安装依赖

#### 管理后台 (homestay-admin)
```bash
cd homestay-admin
npm install
# 或
pnpm install
```

#### 后端服务 (homestay-backend)
```bash
cd ../homestay-backend
npm install
# 或
pnpm install
```

#### 前端应用 (homestay-front)
```bash
cd ../homestay-front
npm install
# 或
pnpm install
```

## 开发命令

### homestay-admin

```bash
# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview

# 类型检查
npm run type-check
```

### 常用开发流程

1. **启动开发服务器**
   ```bash
   cd homestay-admin
   npm run dev
   ```

2. **访问应用**
   - 默认地址: http://localhost:5173
   - 端口可在 `vite.config.ts` 中修改

## 配置说明

### Vite 配置
- 配置文件: `vite.config.ts`
- 支持代理、插件、构建优化等

### TypeScript 配置
- 主配置: `tsconfig.json`
- 应用配置: `tsconfig.app.json`
- 节点配置: `tsconfig.node.json`

### 环境变量
- 创建 `.env` 文件配置环境变量
- 使用 `VITE_` 前缀的变量可在客户端使用

## IDE 配置

### VS Code 推荐插件
- **Vue - Official** - Vue 语言支持
- **TypeScript Vue Plugin (Volar)** - TypeScript 支持
- **ESLint** - 代码规范检查
- **Prettier** - 代码格式化
- **Auto Rename Tag** - 自动重命名标签
- **Bracket Pair Colorizer** - 括号配对高亮

### Cursor / Claude
- 已配置相关规则和优化设置

## 常见问题

### Q: 端口被占用怎么办？
A: 在 `vite.config.ts` 中修改 `server.port` 配置

### Q: 如何配置 API 代理？
A: 在 `vite.config.ts` 的 `server.proxy` 中配置

### Q: 构建失败如何处理？
A: 检查 TypeScript 类型错误和依赖是否完整安装

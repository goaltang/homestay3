# 房源智能特色分析体系

## 系统概述

房源智能特色分析系统是一个多维度、智能化的特色推荐引擎，通过分析房源的各项属性、历史数据、用户反馈等信息，自动生成最多 6 个最具吸引力的特色标签，帮助用户快速了解房源特点。

## 分析维度详解

### 1. 🏠 房源类型与空间分析 (Property Type & Space)

**优先级：11（最高）**

根据房源类型和容纳人数生成基础特色：

- **传统民居**：体验当地风情，区分宽敞/温馨
- **现代公寓**：便利设施齐全，区分家庭式/标准
- **豪华别墅**：私人奢华体验，区分派对/常规别墅
- **特色住宿**：独特体验，留下难忘回忆

### 2. 💰 价格竞争力分析 (Price Competitiveness)

**优先级：9-12（根据竞争力调整）**

分析房源价格在同类型房源中的竞争力：

- **超值选择**：价格低于平均 25%以上，极具性价比
- **价格优势**：价格低于平均 12-25%，性价比很高
- **价格竞争力**：价格低于平均 3-12%，价格有优势
- **市场价格**：价格在 ±8%范围内，符合市场水平

**技术特色**：

- 2 个以上可比房源即可分析
- 容纳人数 ±2 人匹配范围
- 支持同区域/同城市对比

### 3. 🔗 设施组合特色 (Combination Features)

**优先级：7-10**

将多个相关设施组合成更有吸引力的特色：

- **家庭友好厨房**：厨房+微波炉+冰箱
- **便捷出行**：停车位+机场接送
- **宠物天堂**：宠物友好+花园/阳台
- **远程办公**：WiFi+工作区/书桌
- **全方位服务**：清洁服务+洗衣服务
- **贴心服务**：早餐+机场接送
- **娱乐休闲**：游戏设施+影音设备

### 4. ⭐ 单一设施特色 (Key Amenities)

**优先级：7-9**

重点突出核心设施（排除已组合的设施）：

- 优先展示**特殊类别**和**厨房类别**设施
- 过滤已在组合特色中使用的设施
- 根据设施重要性和稀缺性排序

### 5. 📍 地理位置优势 (Location Advantages)

**优先级：8**

分析房源地理位置特点：

- **市中心位置**：交通便利，生活便捷
- **景区周边**：旅游便利，风景优美
- **特色地区**：根据区域名称智能识别特色

### 6. 🔥 预订活跃度分析 (Booking Activity)

**优先级：6-9**

基于近期预订数据生成活跃度特色：

- **近期热门**：过去 90 天内 ≥2 笔预订
- **经验房源**：累计 ≥5 笔订单
- **新推房源**：累计 1-4 笔订单

### 7. 📅 周末偏好分析 (Weekend Popularity)

**优先级：5**

分析房源的周末预订偏好：

- **周末优选**：周五/周六入住比例 ≥30%
- 基于过去 12 个月的预订数据
- 最少需要 3 笔预订记录

### 8. 💬 用户评价分析 (Review Analysis)

**优先级：7-10**

基于用户评价生成特色：

- **广受好评**：平均评分 ≥4.0（最少 3 条评价）
- **关键词特色**：提取积极评价中的高频关键词
  - 支持关键词：干净、位置好、房东热情、设施齐全、安静、舒适、便利、性价比高

### 9. 🎯 搜索匹配优化 (Search Criteria Boost)

**优先级加成：+50**

根据用户搜索条件提升相关特色优先级，增强个性化推荐效果。

### 10. ⚖️ 动态权重系统 (Weight System)

根据房源特性动态调整特色权重：

**基础权重**：

- 房源类型：1.0
- 价格竞争力：1.0
- 组合特色：0.9
- 单一设施：0.7
- 地理位置：0.8
- 预订活跃度：0.6
- 用户评价：0.8

**上下文加成**：

- 房源类型匹配
- 价格区间适配
- 城市等级调整
- 季节性因素

## 技术特点

- 房源特性适配
- 实时动态调整

## 分析流程

```
1. 房源基础信息分析 (房源类型、空间)
2. 价格竞争力分析 (市场对比)
3. 设施组合分析 (多设施组合)
4. 核心设施分析 (单一重点设施)
5. 地理位置分析 (位置优势)
6. 入住便利性分析 (待完善)
7. 预订活跃度分析 (历史数据)
8. 基础活跃度分析 (兜底机制)
9. 周末偏好分析 (时间特征)
10. 用户评价分析 (口碑分析)
11. 搜索匹配优化 (个性化)
12. 权重系统应用 (最终调整)
```

## 优化亮点

### 2024 年 6 月优化

- **降低价格分析门槛**：可比房源数量 3→2
- **放宽价格标准**：竞争力门槛 5%→3%
- **扩大价格覆盖**：新增"市场价格"特色
- **修复类型错误**：解决 HomestayStatus 枚举类型问题
- **提升触发概率**：价格特色更容易出现

### 系统优势

- **全面覆盖**：从基础属性到用户反馈的全方位分析
- **智能权衡**：避免单一维度特色过度集中
- **动态适应**：根据房源特性和用户需求自动调整
- **可扩展性**：模块化设计，便于添加新的分析维度

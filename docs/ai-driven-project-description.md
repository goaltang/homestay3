# 项目成果描述（AI 驱动开发）

## 版本一：简洁版（约 200 字）

本项目是一个基于 AI 辅助编程（Vibe Coding）从零构建的民宿预订全栈平台。开发过程中未编写传统意义上的手写代码，全程由 Cursor、Claude、Codex、Kimi 等十余种 AI 工具辅助完成，人类仅负责需求决策与代码审查。

系统采用 Vue 3 + Spring Boot 技术栈，实现了用户端、房东端、管理员端三端一体，涵盖 Elasticsearch 房源搜索、智能个性化推荐引擎、动态定价引擎、优惠券营销系统、订单自动化状态机、用户行为追踪与画像、支付宝支付等完整业务链路。项目展示了当前大模型在复杂业务系统构建中的能力边界，也验证了非全职开发者借助 AI 工具独立完成生产级系统的可行性。

## 版本二：详细版（约 600 字）

### 项目概述

本项目是一个民宿预订综合业务平台，覆盖房源发布、平台审核、在线搜索、下单支付、入住退房、评价反馈、收益统计和后台监管等完整流程。项目的独特之处在于：**从前端界面到后端架构，从数据库设计到算法实现，全部通过 AI 辅助编程（Vibe Coding）完成**。

### 开发模式

作为非互联网行业从业者，我在 2025 年 2 月至 2026 年期间，系统性地尝试了 Cursor、Claude Code、GitHub Copilot/Codex、Kimi Code CLI、Qwen、Gemini/Antigravity、DeepSeek 等国内外十余种 AI 编程工具。在不具备全职开发背景的前提下，依靠 AI 工具链独立完成了一款包含复杂业务逻辑和算法模块的全栈系统。

### 技术架构

- **前端**：Vue 3 + TypeScript + Vite + Element Plus + Pinia + 高德地图 + ECharts
- **后端**：Spring Boot 3 + Java 17 + Spring Security + JWT + Spring Data JPA
- **数据层**：MySQL 8.0 + Redis + Redisson + Flyway + Elasticsearch
- **通信与集成**：WebSocket（STOMP）、支付宝 SDK、SMTP

### 核心成果

**搜索与推荐**
- 基于 Elasticsearch 构建房源搜索引擎，支持全文检索、地理坐标索引与增量同步
- 实现智能个性化推荐引擎，包含热门推荐、个性化推荐、基于位置的推荐、相似房源推荐四种策略，融合用户历史订单、收藏行为与浏览轨迹构建用户画像

**定价与营销**
- 构建多维度动态定价引擎，支持周末溢价、节假日调价、连住折扣、提前预订优惠等规则，作用域覆盖全局至单个房源级别
- 实现优惠券营销系统，包含模板管理、悲观锁防超领、订单核销、批量发放与 ROI 分析

**订单与状态管理**
- 设计完整的订单状态机与自动化流转机制，基于定时任务实现自动入住、退房、超时取消
- 实现 Redis 分布式锁（Lua 脚本原子解锁 + 故障降级）保障并发场景数据安全

**数据分析**
- 开发用户行为追踪系统，异步采集搜索、浏览、点击、收藏、预订、分享等事件，定时聚合用户画像反哺推荐引擎
- 实现房源智能特征分析与价格竞争力分析，从 8 个维度自动生成房源特色标签

### 项目意义

本项目不仅是一个功能完整的业务系统，更是一次对 AI 编程工具能力的系统性验证。代码中存在的"过度工程化"痕迹（如超出实际需求的定价规则粒度、多维度用户画像分析），恰恰反映了大模型在复杂系统设计中的能力边界与典型特征。

## 版本三：数据驱动版（约 400 字，适合简历）

**民宿预订平台 | 个人项目 | 2025.02 - 至今**

一个通过 AI 辅助编程（Vibe Coding）独立构建的民宿预订全栈平台，采用 Vue 3 + Spring Boot 技术栈，实现用户端、房东端、管理员端三端一体。

**搜索与推荐**
- Elasticsearch 房源搜索引擎：支持全文检索、条件筛选、地理坐标搜索、增量同步与全量重建
- 智能推荐引擎：4 种推荐策略（热门/个性化/位置/相似），3 级降级兜底，基于用户订单+收藏+浏览构建画像

**业务引擎**
- 动态定价引擎：支持周末/节假日/连住/提前预订等多维规则，作用域覆盖全局→城市→房东→房源组→单房源
- 优惠券系统：模板管理 + 悲观锁防超领 + 核销 + 批量发放 + ROI 分析
- 订单状态机：定时任务驱动自动入住/退房/超时取消，支持历史订单批量修复

**基础设施**
- Redis 分布式锁：Lua 脚本原子解锁，Redis 故障时自动降级无锁模式
- 用户行为追踪：异步埋点 6 类事件（搜索/浏览/点击/收藏/预订/分享），独立事务不影响主业务
- 支付宝支付：页面跳转支付 + 扫码支付 + 异步回调 + 退款，完整接入沙箱环境

**开发过程**
系统性地评估了 Cursor、Codex、Kimi、Claude Code、Qwen、DeepSeek 等 10+ 种 AI 编程工具的实际开发能力，最终形成个人工具选型评估体系。
